How to publish as a GitHub Pages project site

  1. Place this index.html at the root of your repository (e.g., shuttle-app/).
  2. In GitHub → Settings → Pages → Source: Deploy from a branch, Branch: main, Folder: / (root).
  3. Your URL will be https://YOURUSERNAME.github.io/shuttle-app/.

Remember: use relative paths like styles/main.css (not /styles/main.css) so assets work under /shuttle-app/.
